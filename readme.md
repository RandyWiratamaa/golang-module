## Membuat module baru pada go
Menggunakan perintah
```
go mod init nama-module
```
Membuat versi tag pada git
```
git tag v1.0.0
```
```
git push origin v1.0.0
```

## Menambahkan Modul untuk Assertions Test
```
go get github.com/stretchr/testify
```
